## Quick Rules

- **Grilled/steamed over fried** — saves 10-20g fat per meal
- **Skip creamy sauces** — ask for sauce on the side
- **Sashimi = purest protein** — best option in any food court
- **Avoid coconut curries** — adds 15-25g fat per serve
